## Плюрализация в русском — важно

Русский язык имеет 4 формы множественного числа (CLDR):
- `=0` → "Нет событий"
- `=1` / `one` → "1 событие"
- `few` (2-4) → "2 события"
- `many` (5+) → "5 событий"
- `other` → запасной вариант

**Всегда** использовать `intl` для плюрализации:

```dart
// ✅ ПРАВИЛЬНО
Text(AppLocalizations.of(context)!.eventsCount(count))

// ❌ НЕПРАВИЛЬНО — ручная логика
Text(count == 1 ? 'событие' : 'событий')
```

## Настройка l10n в Flutter

`l10n.yaml` в корне проекта:
```yaml
arb-dir: lib/l10n
template-arb-file: app_en.arb
output-localization-file: app_localizations.dart
output-class: AppLocalizations
nullable-getter: false
```

`MaterialApp`:
```dart
MaterialApp(
  localizationsDelegates: AppLocalizations.localizationsDelegates,
  supportedLocales: AppLocalizations.supportedLocales,
  locale: ref.watch(localeProvider), // из Riverpod, берётся из users.preferred_language
  // ...
)
```

## Выбор языка пользователем

```dart
// LocaleNotifier (Riverpod)
@riverpod
class LocaleNotifier extends _$LocaleNotifier {
  @override
  Locale build() {
    // 1. Берём из профиля пользователя
    final userLang = ref.watch(currentUserProvider).value?.preferredLanguage;
    if (userLang != null) return Locale(userLang);

    // 2. Fallback на язык устройства
    final deviceLocale = PlatformDispatcher.instance.locale;
    const supported = ['en', 'ru', 'zh'];
    if (supported.contains(deviceLocale.languageCode)) return deviceLocale;

    // 3. Финальный fallback
    return const Locale('en');
  }
}
```

## Текст push-уведомлений

Формируется на сервере (Edge Function) на `users.preferred_language`.
Не локализовать на клиенте — пуш приходит когда приложение закрыто.

Поддерживаемые языки в Edge Function: `en`, `ru`, `zh`.
Fallback: `en`.
