# Экраны приложения

## Навигация (go_router)

```
/                   → SplashScreen
/onboarding         → OnboardingScreen
/login              → LoginScreen
/signup             → SignupScreen (шаг 1/3)
/signup/profile     → ProfileSetupScreen (шаг 2/3)
/signup/notifs      → NotificationSetupScreen (шаг 3/3)
/allset             → AllSetScreen
/forgot-password    → ForgotPasswordScreen
/home               → HomeScreen (ShellRoute с нижней навигацией)
  /home/groups      → GroupsListScreen
  /home/calendar    → CalendarScreen
  /home/timers      → TimersScreen
  /home/profile     → ProfileScreen
/group/:id          → GroupDetailScreen
/group/new          → CreateGroupScreen
/group/join         → JoinGroupScreen
/join/:code         → JoinGroupScreen (deeplink)
/event/new          → EventEditorScreen (создание)
/event/:id/edit     → EventEditorScreen (редактирование)
/event/:id          → EventDetailScreen
/settings           → SettingsScreen
/settings/notifs    → NotificationSettingsScreen
/settings/language  → LanguageSettingsScreen
/settings/timezone  → TimezonePickerScreen
/settings/account   → AccountSettingsScreen
```

---

## Обязательные состояния каждого экрана

Каждый экран с данными ДОЛЖЕН обрабатывать:

| Состояние | UI |
|-----------|-----|
| `loading` | Скелетон или `CircularProgressIndicator` |
| `empty` | Иллюстрация + текст + CTA кнопка |
| `error` | Сообщение + кнопка Retry |
| `offline` | Баннер сверху + данные из кэша |
| `data` | Основной контент |

## Экраны: подробное описание

### SplashScreen
- Лого + анимация
- Проверяет токен авторизации
- Если токен есть → `/home`
- Если первый запуск → `/onboarding`
- Иначе → `/login`

### OnboardingScreen (3 слайда)
- Слайд 1: иконка события + "Не пропускай события альянса"
- Слайд 2: иконка группы + "Создай альянс, пригласи соратников"
- Слайд 3: иконка колокольчика + "Напоминания за 5, 15 или 60 мин"
- Кнопки: "Пропустить" (→ `/login`) и "Далее" / "Начать" (→ `/login`)
- Индикатор прогресса (3 точки)

### LoginScreen
- Email + Password поля
- Кнопка "Войти"
- Ссылка "Забыл пароль?" → `/forgot-password`
- Кнопка "Гостевой режим" → `/home` (ограниченный доступ)
- Ссылка "Зарегистрироваться" → `/signup`

### SignupScreen (шаг 1/3)
- Прогресс-бар: 33%
- Email + Password + Повтор пароля
- Валидация: email формат, пароль ≥ 8 симв, совпадение паролей
- Кнопка "Далее" → `/signup/profile`

### ProfileSetupScreen (шаг 2/3)
- Прогресс-бар: 66%
- Аватар: круг с инициалом + кнопка "Добавить фото"
- IGG ID (обязательное) + подсказка под полем: "Меню → Профиль → Личный ID"
- Ник (обязательное)
- Имя (необязательное) — бейдж "необязательно"
- Часовой пояс: авто-определён + кнопка изменить → `/settings/timezone`
- Кнопка "Далее" → `/signup/notifs`

### NotificationSetupScreen (шаг 3/3)
- Прогресс-бар: 100%
- Toggle: Push-уведомления
- Toggle: Звук
- Toggle: Тихие часы (при включении — поля от/до)
- Чипы интервала (мультивыбор): 5 мин / 15 мин / 60 мин / 3 ч
- Карточка-предупреждение: "Нужно разрешение на уведомления" + кнопка "Разрешить"
- Кнопка "Готово, начать!" → `/allset`

### AllSetScreen
- Анимация успеха (Lottie или кастомная)
- "Ты в игре, {nickname}!"
- Карточка "Что дальше": 3 пункта с галочками
- Кнопка "Создать группу" → `/group/new`
- Кнопка "Войти в существующую" → `/group/join`

### ForgotPasswordScreen
- Экран 1: поле Email + "Отправить ссылку"
- Экран 2 (после отправки): "Письмо отправлено на {email}" + "Вернуться ко входу"
- Deeplink в письме → экран смены пароля (Supabase Auth обрабатывает)

### HomeScreen
- AppBar: ник пользователя + текущая дата + кнопка уведомлений (Phase 2)
- Оффлайн-баннер (если нет сети)
- Секция "СКОРО": события 24 ч вперёд. Каждая карточка:
  - Цветная точка (красная = <30 мин, янтарная = <2ч, синяя = позже)
  - Название + время + бейдж группы/личное
  - Таймер "через X мин" для ближайшего
- Секция "ТАЙМЕРЫ": список активных таймеров HH:MM:SS
- Нижняя навигация: Главная · Группы · Календарь · Таймеры · Профиль

### GroupsListScreen
- Список карточек групп (название, кол-во участников, ближайшее событие)
- FAB или кнопки: "Создать группу" / "Войти в группу"

### GroupDetailScreen
- Заголовок: аватар группы + название + код (копировать)
- Участники: список (аватар + ник + IGG ID + роль)
- Ближайшие события группы (первые 3)
- Кнопка "+ Создать событие" → `/event/new?groupId=...`
- Кнопка "..." меню → покинуть / удалить (только leader) / настройки

### CreateGroupScreen
- Название (обязательное)
- Описание (необязательное)
- Кнопка "Создать" → создаёт группу + генерирует invite_code → `/group/:id`

### JoinGroupScreen
- 3 равнозначных варианта:
  1. По коду: поле 6 симв. + кнопка "Найти"
  2. По ссылке: поле URL + кнопка "Вступить"
  3. Поиск: поле названия + результаты
- После ввода кода — превью-карточка группы (название, кол-во участников) + кнопка "Вступить"
- После вступления → `/group/:id`

### EventEditorScreen (создание и редактирование)
- Название (обязательное)
- Дата и время (picker, с отображением TZ пользователя)
- Описание (необязательное)
- Повторение: чипы Разово / Еженедельно / Ежемесячно
- Напомнить за: чипы мультивыбор 5 / 15 / 60 мин / 3 ч
- Группа: выбор из списка групп или "Личное"
- Видимость: "Вся группа" / "Только я"
- Toggle "Сохранить как шаблон"
- Кнопка "Создать событие" / "Сохранить изменения"

### EventDetailScreen
- Полный заголовок + описание
- Время в локальном TZ пользователя
- Информация о повторении (если есть)
- Напоминания: список интервалов
- Группа или "Личное"
- Кнопки: "Редактировать" (creator/leader) · "Удалить" · "Поделиться"

### CalendarScreen
- Мини-календарь месяца. Точки под датами = есть события
- Список событий выбранного дня:
  - Цветная вертикальная полоска слева (тип события)
  - Время (локальный TZ) · Название · Бейдж группы
- Фильтр: Все / Личные / По группе (dropdown)
- Кнопка "+" → `/event/new`

### TimersScreen
- Список таймеров. Формат: `D д HH:MM:SS` при >24ч, иначе `HH:MM:SS`
- Фильтр: Мои / Группы / Все
- Каждый таймер: название + группа/личный + обратный отсчёт + статус
- FAB → создать таймер
- По истечении: строка окрашивается красным + текст "Завершён"

### ProfileScreen
- Аватар + ник + IGG ID + email
- Кнопка "Редактировать профиль"
- Ссылка "Настройки" → `/settings`

### SettingsScreen
- Секция Профиль (IGG ID, ник, аватар)
- Секция Уведомления → `/settings/notifs`
- Секция Регион: Часовой пояс → `/settings/timezone`, Язык → `/settings/language`, 12/24ч
- Секция Интерфейс: тема, размер шрифта
- Секция Мои группы: список + покинуть
- Секция Аккаунт: email, пароль, выйти, удалить аккаунт
- О приложении: версия, Privacy Policy, Обратная связь
